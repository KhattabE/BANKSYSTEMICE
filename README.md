This project is for our last SP which is called ICE. We have chosen a mobile app which is a banking system with an ATM.
The purpose is to get a better experience with OOP with clear and understandable structure.
The application also has an earluy version of graphical user interface, which we tried for the first time.

Main:
The main class has only one purpose which is to start the system.

BankMenu:
Handles the main user menu and controls navigation between the different functionalities.
Its where most of the application operates .

UIText:
Its responsible for displaying text, messages, and prompts to the user.

User:
Creates a user of the banking system and stores user-related information such as address, phone etc.

Account:
This represents a bank account and manages the account balance.

Saldo:
This handles retrieval and display of the current account balance.

Deposit:
Handles deposit operations and updates the account balance accordingly.

Withdrawel:
Rhis handles withdrawal operations and ensures sufficient balance before withdrawal.

Transactions:
Stores and manages transaction records for deposits and withdrawals.

Database:
Manages the connection between the application and the database.
